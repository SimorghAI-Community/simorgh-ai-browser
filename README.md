# Simorgh AI Research Browser

An open-source AI-powered research browser focused on Claude integration, Persian language intelligence, and next-generation web experiences.

## About

Simorgh AI Research Browser helps users browse, understand, organize, and interact with web content through AI. It combines an extensible browser core with an AI agent system, on-device memory, and native Persian-language intelligence.

## Project Status

The project is currently in early development (Phase v0.1 — Browser Foundation). See [docs/roadmap.md](./docs/roadmap.md) for the full development roadmap.
